A validação visual do certificado no navegador não pôde ser concluída com sucesso porque o módulo http.server do Python (utilizado no Exercício 7) não foi configurado para carregar os arquivos aluno.crt e aluno.key gerados anteriormente. Consequentemente, o servidor não envia o certificado para o navegador, resultando em um erro de protocolo SSL (SSL_PROTOCOL_ERROR).

Caso o servidor estivesse configurado corretamente para responder via HTTPS, o navegador interceptaria o certificado. No entanto, por se tratar de um certificado auto-assinado (Self-Signed) e não validado por uma Autoridade Certificadora (CA) global, o navegador exibiria uma tela de alerta de segurança (Ex: 'Sua conexão não é particular'), exigindo que o usuário aceitasse o risco manualmente para prosseguir.
