O SCP é um protocolo de rede baseado no SSH (Secure Shell) que permite a transferência de arquivos entre diferentes computadores de forma segura. Diferente do FTP tradicional, que envia arquivos em texto claro, o SCP criptografa tanto o arquivo (neste caso, o aluno.crt) quanto as senhas de autenticação durante o tráfego na rede. É uma ferramenta indispensável para realizar o deploy seguro de chaves e certificados da máquina local de um desenvolvedor diretamente para a infraestrutura de produção.
Nesse caso, o comando SCP foi estruturado corretamente simulando a transferência para o localhost usando o usuário root, mas a transferência final foi recusada pela ausência de um servidor SSH ativo na máquina virtual de testes. A lógica de destino e criptografia do protocolo foi devidamente aplicada.
